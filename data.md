## 数据来源

| 数据类型 | 来源 | 覆盖指数 | 历史长度 |
|----------|------|----------|----------|
| PE-TTM | 乐咕乐股(legulegu) + 东方财富 | 全部4只指数 | 2020年起 |
| PB | 乐咕乐股(legulegu) + 东方财富 | 全部4只指数 | 2020年起 |
| 股息率(近期) | 中证指数官网 + 红利查 | 中证红利、上证红利、红利低波 | 近3个月(精确值) |
| 股息率(历史) | 估算：用近期股息支付率 ÷ 历史PE-TTM | 中证红利、上证红利、红利低波 | 2020年起(估算值) |
| 行情(收盘价) | 东方财富 / 中证指数 | 全部4只指数 | 2020年起 |
| 10Y国债收益率 | akshare/中国债券信息网 | — | 2020年起 |

> ⚠️ 中证指数官网导出的估值 Excel 目前通常只保留近 20 个交易日，因此脚本会优先取官网数据，再用红利查详情页内嵌的 DP2/PE2 图表补足到近 3 个月。
>
> ⚠️ **股息率历史数据为估算值**：对于缺少精确股息率的历史区间，用近期 20 个交易日的**股息支付率中位数**（`股息率2 × PE2`）除以当日的 `PE-TTM` 反推得出，历史 PE-TTM 来自中证官网 `/csindex-home/perf/indexCsiDsPe`，公式为 `估算股息率 = 股息支付率 / PE-TTM × 100%`。这假设股息支付率在中期内相对稳定，实际值会有偏差，仅供参考。
>
> ⚠️ 深证红利(399324)当前在中证指数官网估值导出返回 404，红利查也未提供对应页面，因此 `dividend_yield` 仍暂时缺失；后续如找到稳定来源再继续回填。

## 字段说明

| 字段 | 含义 | 单位 |
|------|------|------|
| `date` | 日期 | YYYY-MM-DD |

| `dividend_yield` | 股息率 | % |
| `bond_yield` | 10年期国债收益率 | % |

## 本地使用

**安装依赖：**
```bash
pip install akshare pandas matplotlib
```

**回填历史数据（首次运行）：**
```bash
python scripts/update_data.py --backfill
```

**增量更新（日常使用）：**
```bash
python scripts/update_data.py --incremental
```

**仅生成图表：**
```bash
python scripts/update_data.py --charts
```

**补齐历史股息率估算值：**
```bash
python scripts/estimate_dy_history.py --all
```

---

## 注意事项

1. **周末/节假日无数据**：A股休市日无数据，脚本会自动跳过
2. **数据源不稳定**：akshare接口偶尔改版，如失败请检查日志
3. **分位数计算**：需要至少30个数据点才计算分位数，新数据初期可能显示N/A
4. **股息率/国债比**：>2.5x时配置价值高，<1.5x时吸引力下降
