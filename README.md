## 投资理念

1. **红利 = 价值 + 质量**：持续分红的企业通常盈利稳定、现金流健康
2. **股息率加权**：高分红、低估值股票权重更高，天然低买高卖
3. **长期持有**：红利策略需要时间发酵，不适合短线
4. **定投 + 估值择时**：PE/PB 处于历史低位时加仓，高位减少投入
5. **分散风险**：不押注单一指数，中证红利 + 红利低波组合持有
